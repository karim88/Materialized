KMaterialized WP

This is a simple material design Wordpress template, with thus features:

  • Two-columns
  • Right sidebar
  • RTL support
  • Responsive
  • Featured Image
  • Translation ready
  • Material Design

Note !!!

Is better to use Responsive Lightbox Plugin as required for better experience with image.
